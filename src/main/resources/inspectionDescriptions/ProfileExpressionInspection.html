Reports incorrect @Profile expressions:

  • Infra profiles must not be empty
  • '&' and '|' operators must not be mixed without parentheses in Infra profile expressions

Examples:


  
  @Configuration
  @Profile("a & (b | c)")
  public class MyConfiguration {...}

  
  @Configuration
  @Profile() // reports "Profile expression must contain text"
  public class MyConfiguration {...}

  
  @Configuration
  @Profile("a & b | c") // reports "Malformed profile expression"
  public class MyConfiguration {...}
