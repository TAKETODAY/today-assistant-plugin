Reports invalid prefixes defined in the @ConfigurationProperties annotations:

  • Missing prefix
  • Empty prefix
  • Duplicate prefix
  • Prefix in notation other than kebab-case

The inspection triggers in classes that are annotated with @ConfigurationProperties and not registered via @EnableConfigurationProperties, marked as Infra component, or scanned via @ConfigurationPropertiesScan.

This inspection only triggers in classes annotated with @ConfigurationProperties that are not scanned via @ConfigurationPropertiesScan when the Infra version for the project or module is set to 4.0 or later.
